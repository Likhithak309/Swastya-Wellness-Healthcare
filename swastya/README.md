# 🌿 Swastya Wellness & Healthcare

## Overview

Swastya Wellness & Healthcare is an AI-powered preventive healthcare and wellness platform designed to help users monitor, manage, and improve their overall well-being. The platform provides health guidance, wellness tracking, and essential healthcare resources through an intuitive and user-friendly interface.

The project promotes preventive healthcare by offering tools that encourage healthy lifestyle habits, mental wellness, fitness, hydration, sleep management, and access to healthcare information.

## 🚀 Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/your-username/swastya-wellness-healthcare.git
cd swastya-wellness-healthcare
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

### Open in Browser

```text
http://127.0.0.1:5000
```
